
MULTI-BAND TERAHERTZ ON-CHIP ANTENNA FOR WIRELESS
NETWORK-ON-CHIP (WiNoC) COMMUNICATION
----------------------------------------------------------

Project Overview
----------------
This project presents the design and analysis of a Multi-Band
Terahertz (THz) On-Chip Microstrip Antenna for Wireless
Network-on-Chip (WiNoC) communication applications.

The antenna is designed to operate in the terahertz frequency
range, enabling high-speed wireless communication between
different processing cores inside integrated circuits.
The design is simulated and analyzed using CST Microwave
Studio.

----------------------------------------------------------
Project Objectives
----------------------------------------------------------

• Design a compact on-chip terahertz antenna.
• Achieve multi-band operation in the THz frequency range.
• Improve impedance matching and radiation characteristics.
• Analyze antenna performance using CST Microwave Studio.
• Support high-speed Wireless Network-on-Chip communication.

Antenna Specifications
----------------------------------------------------------

Antenna Type      : Microstrip Patch Antenna
Operating Band    : Multi-Band Terahertz (THz)
Application       : Wireless Network-on-Chip
Substrate         : Silicon
Feed Method       : Microstrip Line Feed
Simulation Tool   : CST Microwave Studio

----------------------------------------------------------
Performance Parameters
----------------------------------------------------------

The antenna performance is evaluated using:

• Return Loss (S11)
• VSWR
• Gain
• Radiation Pattern
• Surface Current Distribution
• Bandwidth
• Radiation Efficiency
• Input Impedance

----------------------------------------------------------
Project Methodology
----------------------------------------------------------

1. Study literature on terahertz antennas.
2. Select operating frequencies.
3. Calculate antenna dimensions.
4. Design antenna geometry in CST.
5. Simulate electromagnetic performance.
6. Optimize antenna dimensions.
7. Analyze S11, gain, and radiation pattern.
8. Validate the final antenna performance.
